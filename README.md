<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">目标是促进四足（四足）机器人研究、教育和敏捷且经济实惠的四足机器人宠物的工程开发方面的合作，将 STEM 概念带给大众，并激励新人（包括许多儿童和成人）加入机器人人工智能领域革命以创建更多应用程序。</font></font></p>
